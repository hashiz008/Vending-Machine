# Vending-Machine
**[RecursionCS](https://recursionist.io) project2-1 Vending Machine**

## Overview
**好きな数字を押すと商品がスライドしPushボタンを押すと金額、商品名がalrertが表示されるアプリです。dom操作とオブジェクトからなるプロジェクトです。表示を表すmainコンテナと非表示を表すextraコンテナでスライダーを実装しました。**
